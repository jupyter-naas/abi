# specs/

Engineering artifacts that inform the published docs but don't belong inside Docusaurus.

| Folder | Purpose |
|---|---|
| `diagrams/` | Source files for architecture visuals (`.excalidraw`, `.drawio`, `.svg`). Export PNGs into `../site/static/img/`. |
| `rfcs/` | Request-for-Comments documents: proposals and design discussions before a decision is made and recorded as an ADR. |
