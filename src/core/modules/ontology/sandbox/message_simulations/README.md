# Message Simulations

This folder contains simulated message exchanges generated by the Message Simulation CLI tool. These simulations are designed to test and validate messaging ontologies with realistic conversation data.

## File Structure

Each simulation generates two files:
- `YYYYMMDD_HHMMSS_topic.md` - Structured markdown with conversation details
- `YYYYMMDD_HHMMSS_topic.csv` - CSV format for data analysis

## Markdown File Contents

### Metadata
- Generation timestamp
- Topic and scenario type
- Complexity level
- Special context/constraints

### Participants
- List of conversation participants
- Participant types (human/AI agent)
- Assigned roles

### Statistics
- Total message count
- Human vs AI message distribution
- Average message length

### Conversation
- Complete message exchange
- Timestamps and sender information
- Message type classification
- Content with proper formatting

## CSV File Structure

| Field | Description |
|-------|-------------|
| message_id | Sequential message identifier |
| sender | Participant name |
| sender_type | human or ai_agent |
| timestamp_offset | Time from conversation start |
| content | Message text content |
| message_type | text, question, answer, clarification, tangent |
| content_length | Character count |

## Usage

Generated via the `messages_simulation.py` CLI tool in the sandbox directory:

```bash
cd src/core/modules/ontology/sandbox
python messages_simulation.py --api-key YOUR_OPENAI_KEY
```

Or set environment variable:
```bash
export OPENAI_API_KEY=your_key_here
python messages_simulation.py
```

## Ontology Testing

These simulations provide realistic data for:
- Message transmission and reception process modeling
- Agent role validation (sender/receiver)
- Content type classification
- Response generation pattern analysis
- Temporal relationship testing
- Multi-participant conversation flows

Perfect for testing the Message Communication Ontology framework!
